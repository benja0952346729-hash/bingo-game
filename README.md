🎮 Online Bingo Game

This is a Telegram Mini App + Bot for playing bingo online.

🚀 Features

- Users can join game
- Real-time bingo board
- Telegram bot integration

📂 Project Structure

- /bot → Telegram bot backend
- /web → Frontend (Mini App)
- .env → Secrets
- package.json → Dependencies

⚙️ Setup

npm install
npm start

🔐 Environment Variables

Create ".env" file:

BOT_TOKEN=your_telegram_bot_token

🌍 Deploy

You can deploy using:

- Vercel (web)
- Render / Railway (bot)
